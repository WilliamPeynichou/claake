## 2. Corrections A11y & Spacing — `frontendApp`

### `index.css`
- `font-size` : 15px → **18px**
- `--color-muted-foreground` : `#6b6558` → **`#766f62`** (ratio 4.5:1 WCAG AA)
- Ajout `:focus-visible` ring vert global `2px solid #2a7a44`
- Ajout `@media (prefers-reduced-motion: reduce)`

### `chat-sidebar.tsx`
- `<div>` → **`<aside>`** (landmark sémantique)
- `role="tablist"` + `role="tab"` + `aria-selected` sur les onglets
- `aria-label` sur tous les boutons icône-only (New, Delete, Sign out, Clear search)
- `aria-current` sur la session active
- Delete button : `group-focus-within:opacity-100` ajouté
- Espacement aéré : header `py-5`, search `py-4`, cards `p-5`

### `chat-messages.tsx`
- `role="log" aria-live="polite"` sur la zone messages
- `role="status" aria-label="Loading"` sur le spinner
- `role="alert"` sur les erreurs
- `space-y-8` entre messages, bulles `px-6 py-4`

### `chat-input.tsx`
- `aria-label` sur textarea et bouton Send
- Focus handlers dupliqués depuis hover
- Container `py-5`

### `chat.tsx`
- `<main>` pour la zone de contenu
- `<header>` pour la top bar
- Top bar `py-5`, description agent `#766f62`

### `login.tsx` / `register.tsx`
- `<main>` wrapper
- `role="alert"` sur les erreurs
- `#a09a8a` → `#766f62` sur tous les textes muted

### `App.tsx`
- `role="status" aria-label="Loading"` sur le spinner de chargement

## 8. Corrections Backend

### `nest-cli.json`
- Builder `swc` → **`tsc`**
- Cause : `@swc/core-darwin-x64` (binding natif) absent de `backend/node_modules/` dans le contexte npm workspaces

### `src/common/guards/roles.guard.ts`
- `import type { Reflector }` → **`import { Reflector }`**
- Cause : `import type` est effacé à la compilation, TypeScript émettait `Function` au lieu de `Reflector` dans `design:paramtypes`, cassant l'injection de dépendances NestJS

### `src/modules/agents/agent.module.ts`
- Ajout de `Reflector` dans `providers[]`
- Import de `Reflector` depuis `@nestjs/core`
