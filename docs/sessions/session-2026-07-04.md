### 2. Fix UX mode test — commit `0f5ec86`

`fix: force chat test mode for unpublished agents when backend allows chat`

  `can_chat: true` mais la création de session échouait ("This agent is not available").

  et `access.can_chat`. Le backend re-vérifie les droits (pas de faille).
- `ChatShell` affiche le bandeau "Mode test" dès que le mode test effectif est actif.
